# Offline Research Verification System

A fully offline, explainable research verification tool that:
- Extracts claims from PDFs
- Verifies them semantically
- Visualizes evidence
- Answers questions from documents
- Generates verification reports

## Features
- No API keys
- No cloud dependencies
- Local ML embeddings
- Interactive graphs
- Multi-PDF support
- Docker deployable

## Local Setup (No venv)

### Requirements
- Python 3.11
- pip
- Docker (optional)

### Install
```bash
pip install -r requirements.txt
